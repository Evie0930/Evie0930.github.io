一、本 ZIP 包内容
--------------------------------------------------------------------------------
  - index.html          单页站点入口（Tailwind CDN + 交互脚本）
  - style.css           全局样式与设计令牌（卡片阴影、展开动画、字体栈等）
  - assets/             静态资源目录
      wechat-qr.png     微信联系弹窗用二维码图
  - 根目录下其它图片文件（若存在）：一并纳入，便于后续替换页面配图引用

解压后请将**全部文件保持相对路径**置于仓库根目录，或 GitHub Pages 所配置的
发布目录（如 / 或 /docs）下，勿改动 index.html 与 style.css、assets 的相对路径。

--------------------------------------------------------------------------------
二、GitHub Pages 部署说明（简要）
--------------------------------------------------------------------------------
  1. 新建仓库或将文件推送到已有仓库。
  2. Settings → Pages → Build and deployment → Source 选择分支与文件夹
     （常见：main 分支 / root，或 main /docs）。
  3. 若站点地址为 https://<user>.github.io/<repo>/（项目页），需在
     index.html 的 <head> 中增加 <base href="/<repo>/"> 或批量改写资源路径；
     用户页 https://<user>.github.io/ 则无需 base，当前相对路径可直接使用。
  4. 本站使用外链：Tailwind CSS CDN、via.placeholder.com 配图；部署需联网加载。

三、功能与结构摘要（历次迭代汇总）
--------------------------------------------------------------------------------
  - 全局导航：品牌、灵感洞察、个人品牌、项目/实习、探索世界、娱乐；全站搜索索引
  - Hero：姓名、标签、简介、Slogan、「点击联系我」微信 Modal
